
-------------------------------------------------------------
The API provided in 'org.toolfactory.java.agreement' package tries to explore/simplify a useful Design Pattern, "Command Design Pattern"	
----------------------------------------------------------------

AgreementTest.java implements this API.

Classes/Interfaces within "org.toolfactory.java.agreement" package are the building block of API.

Classes within "org.toolfactory.java.agreement.impl" package are extreamely simple implementation of Command and Action.
